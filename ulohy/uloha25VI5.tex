Na zaèátku sedí Aleš v~poèátku souøadného systému. Cihlu odhazuje 
v~bodì $(x_0,y_0)=vT(-\cos\alpha,\sin\alpha)$. Alešova rychlost je $v(-\cos\alpha,\sin\alpha)$ 
a~v~jeho sou\-øad\-ném systému bude odhazovat cihlu rychlostí $w(\cos\beta,\sin\beta)$. V~nehybném 
sou\-øad\-ném systému bude tedy odhazovat rychlostí
\eq{
    (w'_x,w'_y)=(w\cos\beta-v\cos\alpha,w\sin\beta+v\sin\alpha)\,.
}

Trajektorie cihly bude
\eq{
    (x,y)(t)=\big(x_0+w'_xt,y_0+w'_yt-\frac12gt^2\big)\,.
}
Naším úkolem je vyøešit soustavu rovnic $(x,y)(t;\alpha,\beta,T,v,w)=0$ pro neznámé $t$ a~$\beta$, 
zatímco $\alpha$, $T$, $v$ a~$w$ jsou parametry.

Z~kvadratické rovnice $y(t)=0$ dostaneme
\eq{
    t=\frac{w'_y}g+\sqrt{\( \frac{w'_y} g \)^2+\frac{2y_0}g}\,.
}
Dosazením do $x(t)=0$ a~rozepsáním $w'_x$ a~$w'_y$ dojdeme
k~\eq[m]{
    -vT\cos\alpha+(w&\cos\beta-v\cos\alpha) \left[ \frac{w\sin\beta+v\sin\alpha}g + 
    \vphantom{\sqrt{\(\frac{w\sin\beta+v\sin\alpha}g\)^2}} \right.\\ % kvùli výšce levé závorky
    & \left. + \sqrt{\(\frac{w\sin\beta+v\sin\alpha}g\)^2+\frac{2vT\sin\alpha}g} \right] = 0\,.
		\lbl{R25S6U5_rovnice}
}
Nìkolika algebraickými úpravami a~schováním $v$, $w$, $g$ a~$T$ do bezrozmìrných $Q=w/v$ a~$A=gT/v$ 
dostaneme
\eq{
    A(1+\cos2\alpha)+4Q\sin(\alpha+\beta)(\cos\alpha-Q\cos\beta)=0\,.
}
Tato rovnice bohužel vede na kvadratickou rovnici v~$\sin\beta$ (nebo $\cos\beta$), jejíž monstrózní 
øešení nám neøekne zhola nic, a~proto se uchýlíme k~numerickému øešení.

Pro tyto úèely je dobré se vrátit k~rovnici~\eqref{R25S6U5_rovnice}, do které jsme ještì nezanesli 
neekvivalentními úpravami nesprávná øešení. Po chvíli hraní s~touto rovnicí v~poèítaèi dojdeme k~závìru, 
že dle oèekávání má Aleš buï smùlu a~nedohodí, a~nebo má na výbìr ze dvou rùzných úhlù $\beta$. Taktéž se 
ukazuje, že pro numerické úèely bude vhodnìjší nehledat $\beta$ v~závislosti na $\alpha$ (0 nebo 2~øešení), 
ale radìji $\alpha$ v~závislosti na $\beta$ (0 nebo 1~øešení). Výsledné grafy (už zase jako $\beta(\alpha)$) 
jsou na obrázku~\ref{R25S6U5_reseni}.
